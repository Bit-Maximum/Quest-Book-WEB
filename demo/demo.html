Поиск артефакта

Ваш путь прегдаждает упавшее посреди дороги огромное дерево. Это одно из первых препядствий на пути к сокровищам Храма Затерянных Тайн. Иного пути к месту раскопок вы не находите, и в тоже время пешком добираться не представляется возможным.

Чтобы продолжить путь, вам нужно решить, как избавится от дерева. Кажется, дерево можно разрубить. Может увас найдётся подходящий для этого инструмент...

Код матрицы

Чтобы проверить свои трехзначные коды (они же «Номера ячеек»), начните в таблице со слова СТАРТ и следуйте инструкциям ниже. Предварительно потренеруйтесь хотя бы один раз, чтобы разобраться в примере кода.

1 ЦИФРА: считать X клеток справа

2 ЦИФРА: считать Y клеток внизу

3 ЦИФРА: в зависимости от цифры выполнить действие, согласно нижнему краю таблицы

Код этого примера: 387

Кот

0 - 3 клетки, для подсчета вниз
1 - 3 клетки, для подсчета влево
2 - 4 клетки, для подсчета вверх
3 - 4 клетки, для подсчета вправо
4 - 2 клетки, для подсчета влево

5 - 3 клетки, для подсчета вверх
6 - 4 клетки, для подсчета влево
7 - 2 клетки, для подсчета вверх
8 - 3 клетки, для подсчета вправо
9 - 2 клетки, для подсчета вниз
